title: 无需任何配置，直接使用sbt运行一个scala文件
---

简单的说，就是在某个目录中创建一个scala文件，直接用sbt执行：

```shell
mkdir mypro
cd mypro
touch Hello.scala
# edit Hello.scala

sbt run
```

当然前提是你已经安装了sbt。

sbt给我的感觉是非常复杂和笨重，没想到还能这么用，感觉还是很方便的。
